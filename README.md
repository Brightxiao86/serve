
# My Express App

## 📌 项目简介

这是一个 Express.js 项目示例，支持快速搭建 Web 服务器。

## 🚀 安装步骤

 1.**安装依赖**

```bash
   npm install
```

2.**启动服务**

```bash
   npm start
   然后访问 [http://localhost:3000]
```

3.**配置表**

```bash
 可以参照  db文件夹中initialization文件 去初始化表如果有表就不创建 否则就创建
```
